types:
    1. Centralized Version control system (CVSC):

    debvelopers works in a single remote repository
    examples: Subversion(SVN), concurrent version system(CVS).

    2. Distributed version control system (DVCS):
    developers have copies of the remote repo in their local machines .
    examples: GIT, mercurial, BitBucket
